# MBTI 성격 유형 검사 서비스
## 프로젝트 설명
단순한 MBTI 테스트에 **회원가입/로그인**, **프로필 관리**, **테스트 결과 저장** 의 기능을 추가하여 만든 프로젝트. **JWT 인증**과 **REST API 통신**을 실습하는 애플리케이션이기도 합니다. (`Axios`, `Tanstack Query(React Query)`, ` json-server` 등을 활용합니다)
